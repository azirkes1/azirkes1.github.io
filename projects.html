Wildfire Management Data App

While interning for The Nature Conservancy, I created a web app using Python and Streamlit to make critical wildfire management data more accessible to landowners in Bristol Bay, Alaska. Users select the data layers and data format they are interested in and draw a boundary on the map. The output is either a tif file or pdf map of the data clipped to their boundary.


Key skills: Python, ArcGIS Pro, automated geoprocessing, data cleaning & visualization, creative problem solving Python modules utilized: streamlit, folium, numpy, rasterio, matplotlib, and geopandas

Burn Severity and Vegetation Recovery Tool

Alongside the Wildfire Management Data App, I also developed Google Earth Engine code to help land managers in Bristol Bay, Alaska assess burn severity and vegetation recovery after wildfires. This code uses Landsat and Sentinel 2 imagery to calculate the differenced Normalized Burn Ratio (dNBR) and the differenced Normalized Difference Vegetation Index (dNDVI). In addition, I recorded a tutorial as this tool is intended for users who have little to no experience with Google Earth Engine.


Key skills: JavaScript, Google Earth Engine, satellite image processing, remote sensing indices, fire ecology

Devon Ice Cap Retreat, Velocity, and Elevation Analysis

As a final project during my master’s coursework, I measured the retreat of Devon Ice Cap with SWIR1 band differencing. I used Landsat imagery and Google Earth Engine for this analysis. I also analyzed global glacier datasets to investigate changes in velocity and elevation on this ice cap and created visualizations in ArcGIS Pro.


Key skills: Google Earth Engine, ArcGIS Pro, Remote sensing indices, and data wrangling, management, & visualization

Click here to read the final report


Retreat of Devon Ice Cap 1987 – 2024. Total retreat measured is 617 km2.
